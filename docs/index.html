📖 Cómo instalar

Para Android (APK)

  1. Descarga el archivo APK desde el botón de arriba
  2. Habilita instalación desde fuentes desconocidas:
    • Ve a Configuración → Seguridad
    • Busca "Instalar apps desconocidas" o "Fuentes desconocidas"
    • Habilita la opción para tu navegador o administrador de archivos
  3. Abre el archivo APK descargado
  4. Toca "Instalar" y espera a que finalice
  5. ¡Listo! La app aparecerá en tu menú de aplicaciones

Para Google Play Store (AAB)

El archivo AAB es para desarrolladores. Usa el APK para instalar directamente en tu teléfono.

⚠️ Nota importante: Si ves una advertencia de "Aplicación desconocida", es normal. Nuestra app es segura y no contiene malware. Simplemente haz clic en "Instalar de todas formas".

❓ Solución de problemas

La app no se instala

Asegúrate de que:

  • Habilitaste "Instalar apps desconocidas"
  • Tienes suficiente espacio en tu teléfono
  • La versión de Android es 6.0 o superior

La cámara no funciona

Verifica que diste permiso de cámara a la app en:

  • Configuración → Aplicaciones → Verifica tu billetito → Permisos
  • Asegúrate que "Cámara" esté habilitada

Otros problemas

Si tienes otro problema, puedes:

  • Reiniciar tu teléfono
  • Desinstalar y reinstalar la app
  • Asegurarte de tener conexión a internet
